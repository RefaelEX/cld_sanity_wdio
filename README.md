## cld_sanity_wdio
Automation tests implementation with WebdriverIO and Allure report frameworks using page object pattern.

#### Prerequisites :
  * Nodejs (tested with version 10.19.0)
  * Java (tested with version 11.19.0)
  * Chrome (tested with version 84)
  
#### Installation : 
  * Run `npm install`

#### Run Tests :
  * Run `npm run sanity`

#### Generate Report :
  * Run `npm run report`
